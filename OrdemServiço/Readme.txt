----- Projeto Oficina Mecânica ----

Projeto esquama lógico do zero para banco de dados de gerenciamento de ordens de serviços para oficina mecânica.Conforme script que envio anexado, apresento as tabelas e relações sugeridas para a criação das tabelas, como também a utilização de constraints e regras para atributos.


Tabelas: Veículo, Equipe, Mecanico, Ordem de Serviço, Peça, Serviço.

Foram persistidos algumas informações para exemplo, e em seguida aprento algumas query para testes das tabelas.
